# **App Name**: CareerPilot AI

## Core Features:

- AI-Powered Resume Analysis: Utilizes a tool to analyze resume content against ATS standards, providing a circular score ring and specific improvement suggestions.
- Adaptive Skill Assessments: Interactive multiple-choice quizzes across various technical stacks with real-time scoring and progress saving.
- Visual Career Roadmaps: Dynamic, timeline-based career paths that track progress status and unlock next steps as skills are mastered.
- Learning Analytics Suite: Comprehensive data visualization dashboard featuring line, bar, radar, and pie charts to track weekly learning trends.
- Digital Certificate Vault: A centralized management system to add, filter, and search through professional certificates using glassmorphism design cards.
- LocalStorage Persistence Engine: Ensures full application state, user profiles, and assessment history are saved and retrieved using local storage mechanisms.

## Style Guidelines:

- Primary Color: Vibrant Purple (#915EFF) inspired by modern SaaS tech themes, providing high visibility in a dark environment.
- Background Color: Deep Charcoal/Indigo (#0D0B14), heavily desaturated to anchor the glassmorphism and gradient effects.
- Accent Color: Electric Blue (#4D79FF), analogous to the primary purple, used for success states and secondary calls-to-action.
- Headline Font: 'Space Grotesk', a tech-focused sans-serif to emphasize the futuristic AI nature of the platform. Body Font: 'Inter' for maximum legibility in data-dense sections.
- Minimalist line icons with subtle purple glows to maintain the high-end digital agency aesthetic.
- Glassmorphism containers with semi-transparent backdrops and 1px borders to separate dashboard cards from the dark background.
- Smooth vertical reveal transitions and gradient-shifted hover states on interactive dashboard elements.
