# CareerPilot AI 🚀

CareerPilot AI is an AI-powered career guidance and learning analytics platform designed to help students track skills, analyze career growth, manage certificates, and build a structured learning roadmap.

## 📌 Features

- AI-Powered Career Dashboard
- Skill Assessment & Progress Tracking
- Learning Analytics with Interactive Charts
- Resume Score Analysis
- Certificate Management System
- Career Roadmap Generator
- Responsive Modern UI
- Dark Theme Design
- Local Storage Data Persistence

## 🛠️ Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Recharts
- Lucide React
- LocalStorage

## 📊 Modules

### Dashboard
- Career score tracking
- Skill overview
- Learning progress

### Analytics
- Weekly learning trends
- Skill proficiency radar chart
- Certification statistics

### Resume Analyzer
- Resume scoring system
- Improvement suggestions

### Certificate Vault
- Store and manage certificates
- Track learning achievements

### Career Roadmap
- Structured learning path
- Goal-based progression

## 📱 Responsive Design

The application is fully responsive and optimized for:

- Desktop
- Tablet
- Mobile Devices

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/careerpilot-ai.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
